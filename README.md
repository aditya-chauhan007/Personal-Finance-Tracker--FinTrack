# FinTrack - Personal Finance Tracker

A full-stack MERN finance tracker with JWT auth, MongoDB persistence, animated React dashboards, budgets, CSV export, currency formatting, and multilingual UI.

## Run locally

1. Run `npm run install:all`.
2. Start MongoDB locally or update `server/.env` with an Atlas URI.
3. Run `npm run dev`.

Frontend: http://localhost:5173
Backend: http://localhost:5000

New accounts start with no seeded data.
